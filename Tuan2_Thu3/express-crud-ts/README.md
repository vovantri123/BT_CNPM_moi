## Running the Project

To clean the build directory and start the application, use the following commands:

```powershell
# Remove the 'dist' directory
Remove-Item -Recurse -Force dist

# Start the application
npm start
```

## Getting Started

1. **Install dependencies:**

    ```bash
    npm install
    ```

2. **Build the project:**

    ```bash
    npm run build
    ```

3. **Start the server:**
    ```bash
    npm start
    ```
